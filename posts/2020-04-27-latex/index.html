Handy links for LaTeX

  • Detexify automatic recognition of handwritten symbols
  • latexpand to automatically combine all latex files into a single file. (If full latex is installed, you just need to type this command line: latexpand -empty-comments -o main_one_file.tex main.tex the long file output will appear in the Terminal)
  • mathcha for creating tikz pictures in a more eye-friendly way
  • FreeTikZ to transform handwritten diagrams into tikz
  • TikZiT GUI editor for graphs and string diagrams
